เกี่ยวกับ

สวัสดีครับ ผม นาย คุณากร ทําดี สำเร็จการศึกษาระดับปริญญาตรี สาขาวิศวกรรมคอมพิวเตอร์ จากมหาวิทยาลัยนครพนม ผมเป็นนักศึกษาจบใหม่ที่มีประสบการณ์ในการพัฒนาเว็บไซต์แบบ Static Website จากการรับงานฟรีแลนซ์ โดยถนัดด้าน Frontend เป็นหลัก โดยใช้ HTML, CSS และ JavaScript ส่วนด้าน Backend ใช้ PHP และ Node.js นอกจากงานเว็บไซต์แล้ว โปรเจกต์จบการศึกษาของผมคือการพัฒนา เกม 3D แนว Base Defense ด้วย Unity Game Engine ซึ่งผู้เล่นรับบทเป็นผู้พิทักษ์ที่ต้องปกป้อง “Energy Pillar” จากการรุกรานของศัตรู เกมนี้มีระบบ การต่อสู้, การป้องกัน, การจำกัดเวลา และการโต้ตอบกับสิ่งแวดล้อม ซึ่งช่วยเพิ่มทักษะการออกแบบระบบ, การใช้ C# ใน Unity และการแก้ไขปัญหาทางเทคนิคให้สมบูรณ์ ผมยังสามารถใช้ภาษา Python, Java และ C++ ได้ พร้อมทั้งเป็นผู้ที่เรียนรู้สิ่งใหม่ ๆ ได้รวดเร็ว ปรับตัวได้ดี และทำงานภายใต้แรงกดดันได้อย่างมีประสิทธิภาพ ผมมีความตั้งใจที่จะสมัครงานในตำแหน่งที่เกี่ยวข้องกับ IT หรือ Web Development เพื่อพัฒนาทักษะและเติบโตไปพร้อมกับองค์กร สำหรับเงินเดือนที่คาดหวังอยู่ที่ 16,000 บาทขึ้นไป ทั้งนี้ยินดีพิจารณาตามโครงสร้างและข้อเสนอของบริษัทครับ
